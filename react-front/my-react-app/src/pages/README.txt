דף אודות (About)
בדף זה אנו מציגים את מטרת הפלטפורמה ואת היתרונות של שימוש בה. אנו מדברים על איך המערכת יכולה לסייע למשתמשים לשפר את הרשת העסקית שלהם.

דף כרטיסי העסקים שלי (BusinessList)
בדף זה, מוצגים כל כרטיסי העסקים שהמשתמש יצר. המשתמש יכול לערוך או למחוק כרטיסים קיימים. הכרטיסים המוצגים שייכים למשתמש הנוכחי בלבד.

 (Favorites)
בדף זה, מוצגים כרטיסי העסקים המועדפים של המשתמש. משתמש יכול לבחור כרטיסים מועדפים ולצפות בהם בלחיצת כפתור.


(BusinessCard)
כרטיס עסק מציג את כל המידע הקשור לכרטיס, כולל תמונה, כותרת, תיאור, מספר טלפון, אפשרות לסמן כרטיסים מועדפים, ולערוך או למחוק את הכרטיס.

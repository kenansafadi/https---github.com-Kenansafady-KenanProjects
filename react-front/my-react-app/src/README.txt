פרויקט React - מערכת ניהול כרטיסי עסק
הפרויקט הוא מערכת ניהול כרטיסי עסק, המאפשרת למשתמשים להירשם, להתחבר, ולנהל את כרטיסי העסק שלהם. המערכת כוללת דפים להצגת כרטיסים, דף פרופיל אישי, דף עלינו, וממשק חיפוש.

טכנולוגיות בשימוש:
React: עבור הפיתוח של רכיבי ממשק המשתמש.
React Router: לניהול ניווט בין דפים.
Context API: לניהול סטייט של המשתמשים והכרטיסים.
API: ממשק לקישור עם שרת חיצוני לצורך שליפת כרטיסי עסקים.
Sass: עבור עיצוב הממשק.
מבנה הפרויקט:
הפרויקט כולל את המבנה הבא:

components: רכיבי UI כמו כרטיסי עסק, חיפוש, תפריט ניווט, רגלי דף.
pages: דפים שונים כמו דף הבית, דף כרטיסי עסקים, דף פרופיל, ועוד.
services: שירותים לקישור עם ה-API לצורך שליפת כרטיסים וניהול משתמשים.
context: לניהול מצבים גלובליים כמו פרטי משתמש והכרטיסים.
utils: פונקציות עזר, כגון עזרה בטיפול בטוקנים.
דפים:
דף הבית: מציג את כרטיסי העסק הזמינים.
כרטיסי עסק: מציג את פרטי כרטיסי העסק.
דף פרופיל: דף בו המשתמש יכול לראות ולעדכן את פרטי המשתמש.
חיפוש: מאפשר חיפוש כרטיסי עסק לפי שם או תיאור.
הרשמה/התחברות: דפים להרשמה והתחברות למערכת.
הפעלת הפרויקט:
התקן את התלויות:
npm install
הפעל את הפרויקט בסביבת פיתוח:
npm run dev

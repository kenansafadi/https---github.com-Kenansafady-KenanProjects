ooter
מציג את קרדיט האתר עם השנה הנוכחית.

ListComponents
מציג אייקונים עבור כרטיסי עסקים, כולל אפשרויות עריכה, מחיקה (עם אישור), ושיחה (אייקון טלפון).

MyComponent
מציג אייקונים לאהוב (לב אדום) ולבצע שיחה (אייקון טלפון), ושומר את המצב (האם נשמר במועדפים או לא).

Navbar
מציג את תפריט הניווט הכולל קישורים לעמודים כמו "B-Card", "About", "My Cards", "Favorites", "Logout", ותמיכה במצב כהה.

SearchBar
סרטון חיפוש שמאפשר למשתמש לחפש כרטיסי עסקים.

UserPlus
אייקון שמאפשר להוסיף כרטיס עסק חדש, תוך הצגת טופס לשם כך.

BusinessDetails.jsx
מציג את פרטי כרטיס העסק, כולל כותרת, תיאור, פרטי יצירת קשר, כתובת עם מפה, וקישור ל-Google Maps.

DeleteBusinessCard.jsx
מכיל כפתור למחיקת כרטיס עסק. בודק אם יש כרטיס ואם הוא מכיל את ה-ID הנדרש לפני מחיקה.

EditBusinessCard.jsx
מראה טופס עריכת כרטיס עסק, כולל שדות כמו כותרת, תיאור, טלפון, דוא"ל, כתובת ועוד. מבצע אימות על השדות ומעדכן את הכרטיס לאחר הגשה.

FavBusinessCards.jsx
מציג כרטיסי עסקים מועדפים. כולל אפשרות לאהוב (לב) ולהסיר את הכרטיס מהמועדפים.
